# Matriz de Risco Jurídico

Este documento define os critérios para a atribuição do placar de risco em documentos jurídicos.

## 🔴 Risco Alto
Um documento é classificado como de Risco Alto se apresentar uma ou mais das seguintes características:
- **Abusividade Clara**: Cláusulas que violam o Código de Defesa do Consumidor (CDC) ou a LGPD.
- **Desequilíbrio Contratual Extremo**: Multas pesadas apenas para uma das partes; rescisão imotivada apenas para o fornecedor.
- **Renovação Automática com Fidelidade**: Cláusulas que renovam o contrato por períodos longos sem aviso prévio e com multas de cancelamento.
- **Coleta Indiscriminada de Dados Sensíveis**: Coleta de dados sem finalidade clara ou compartilhamento com terceiros sem consentimento.
- **Foro de Eleição Impossível**: Eleição de foro em outro país ou cidade de difícil acesso para o consumidor.

## 🟡 Risco Médio
Um documento é classificado como de Risco Médio se:
- **Cláusulas Padrão, mas Onerosas**: Multas dentro dos limites legais, mas que exigem planejamento financeiro.
- **Renovação Automática Transparente**: Prevê a renovação, mas indica claramente como cancelar.
- **Uso de Dados para Marketing**: Coleta dados para melhorar o serviço e marketing, mas oferece opção de opt-out.
- **Arbitragem Obrigatória**: Pode dificultar o acesso à justiça comum, embora seja um método legal.

## 🟢 Risco Baixo
Um documento é classificado como de Risco Baixo se:
- **Equilíbrio entre as Partes**: Direitos e deveres claros e recíprocos.
- **Transparência Total**: Linguagem acessível, sem letras miúdas.
- **Privacidade por Design**: Segue estritamente a LGPD, coletando apenas o mínimo necessário.
- **Rescisão Facilitada**: Possibilidade de cancelar o serviço a qualquer momento sem penalidades abusivas.
